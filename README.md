# bookmarks-server

This project is the server for [the bookmark application](https://github.com/leguillantjulien/bookmarks).

It provides 5 REST API:
- findAll
- findMediaLinkById
- deleteMediaLink
- addMediaLink
- updateMediaLink

# Usage
### Install 
- `git clone https://github.com/leguillantjulien/bookmarks-server.git`
- `cd bookmarks-server`
- `npm install`

### Launch
`npm run start`

### Build
`npm run build`
